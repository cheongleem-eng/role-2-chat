# Project Blueprint: 제주항공 비상훈련 롤플레잉 시스템

## 1. 개요 (Overview)
이 애플리케이션은 제주항공 객실승무원들의 비상상황 대응 능력을 평가하고 훈련하기 위한 시스템입니다. PRAM, Interphone 조작 학습 및 상황별 롤플레잉 시나리오를 제공합니다.

## 2. 상세 구성 (Detailed Outline)

### 스타일 및 디자인 (Style & Design)
- **테마**: 제주항공 브랜드 컬러(주황색)와 어두운 배경의 다크 모드 테마.
- **레이아웃**: 모바일 퍼스트 디자인으로, 상단 헤더, 중앙 콘텐츠 영역(스크롤 가능), 하단 내비게이션 바 구성.
- **컴포넌트**: 카드형 UI, 애니메이션 효과(Blink 등), 이모지 기반 평가 버튼.
- **아이콘**: Phosphor Icons 라이브러리 사용.

### 주요 기능 (Features)
- **포털 화면**: 네 가지 주요 모듈(PRAM, Interphone, Emergency Role-Playing, Real-time Chat) 진입점 제공.
- **로그인 시스템**: 평가자/피평가자 정보 입력 및 Firebase 연동을 통한 세션 관리.
- **롤플레잉 평가**: 단계별(Phase) 시나리오 제시 및 점수 기록 기능.
- **실시간 채팅 (Open Chat)**:
  - 관리자 방 생성 및 **삭제(휴지통 아이콘)** 기능.
  - 입장 전 닉네임 및 조(1조~6조) 설정, 관리자 모드 지원.
  - **메시지 프로필 표시**: "1조: 홍길동" 또는 "관리자" 형식으로 개선.
  - **참여자 목록**: 채팅방 내 실시간 인원 확인 및 명단 보기 기능(👥 아이콘).
- **관리자 대시보드**: 전체 훈련 기록 열람 및 엑셀 다운로드 기능.
- **Firebase 연동**: 데이터 실시간 저장 및 동기화.

## 3. 현재 작업 계획 (Current Task Plan)

### 목표: 오픈 채팅 최종 고도화 (참여자 목록, 방 삭제, 프로필 개선) - 완료 (Completed)
- 메시지 프로필 형식을 "조: 이름"으로 통일하고 관리자는 "관리자"로 표시.
- 관리자 전용 방 삭제(Firestore 문서 삭제) 기능 구현.
- 채팅방 내부 실시간 참여자 수 집계 및 목록 팝업 UI 구현.
